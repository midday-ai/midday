Track your work hours accurately to bill clients and understand where your time goes.

## Start the timer

1. Go to [Time Tracking](https://app.midday.ai/tracker)
2. Select a project
3. Click **Start timer**
4. Work on your task
5. Click **Stop** when done

The timer runs in the background. You can navigate away—it keeps tracking.

## Log time manually

For time you forgot to track:

1. Go to [Time Tracking](https://app.midday.ai/tracker)
2. Click **Add entry**
3. Select the project
4. Enter the date and duration
5. Add an optional description
6. Save

## Time entries

Each entry records:

- **Project**: Which project the time is for
- **Date**: When the work was done
- **Duration**: How long (in hours or hours:minutes)
- **Description**: Optional note about what you worked on

## Edit or delete entries

1. Find the entry in the time tracking list
2. Click to edit
3. Change project, duration, or description
4. Save or delete

## View your time

### By day

See entries grouped by date. Good for reviewing your workday.

### By project

See total time per project. Good for checking progress toward estimates.

### By week or month

Use date filters to see totals for any period.

## Tips for accurate tracking

- **Start the timer immediately**: Even if you're just "checking one thing"
- **Add descriptions**: Future you will thank present you
- **Log forgotten time the same day**: Memory fades quickly
- **Review weekly**: Catch any gaps before invoicing

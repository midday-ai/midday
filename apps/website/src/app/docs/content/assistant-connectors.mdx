The Midday assistant can interact with external services you connect — reading data, taking actions, and pulling context from the tools you already use, all from the chat.

## How it works

1. You connect an app from the Connectors settings in your dashboard
2. The assistant automatically discovers what that app can do
3. When your question or request involves that app, the assistant uses it

No API keys or manual configuration required. Authentication is handled through the Connectors UI.

## Connecting an app

1. Open [Midday](https://app.midday.ai)
2. Go to **Apps** in the sidebar
3. Browse available connectors or search by name
4. Click **Connect** and authenticate with the service
5. The app is now available to the assistant

## Disconnecting an app

1. Go to **Apps** in the sidebar
2. Find the connected app
3. Click **Disconnect**

The assistant will no longer have access to that service. No data from the service is stored in Midday after disconnection.

## Security and privacy

- Each connection uses secure OAuth — Midday never stores your passwords for connected services
- The assistant only accesses data when you ask it to in a conversation
- You control which apps are connected and can disconnect at any time
- Connected app data is not used to train AI models
